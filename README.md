# Analyzing Movie Ratings and Trends Using EDA

## Project Overview
This project focuses on analyzing movie and TV show ratings data using exploratory data analysis (EDA) techniques to identify content trends, genre popularity, and audience preferences.

## Tools Used
- Canva (Data Visualization & Presentation)
- Python (EDA concepts)
- MS Excel / CSV Dataset

## Dataset
Publicly available movie and TV show dataset related to streaming platforms.

## Key Analysis Performed
- Genre-wise content distribution
- Ratings trend analysis
- Release year vs rating patterns
- Country-wise content availability

## Key Insights
- Drama and Comedy dominate content libraries
- Recent releases show higher audience engagement
- Certain regions produce higher-rated content

## Outcome
The project demonstrates the ability to interpret datasets, visualize insights clearly, and communicate findings effectively for data-driven decision-making.
